Main page

res.redirect: 메인 페이지로 바로 리다이렉트됩니다.

res.render: 백엔드에서 정보를 받아서 페이지를 랜더합니다.

res.send 1: 백엔드에서 바로 html 코드를 보내서 실행합니다.

res.send 2: 백엔드에서 바로 script 코드를 보내서 실행합니다.

[form] req.body: body에 정보를 담아서 백엔드로 보냅니다.

[board] req.params: path에 정보를 담아서 백엔드로 보냅니다.

[search] req.query: 쿼리스트링에 정보를 담아서 백엔드로 보냅니다.

body, params, query 세 가지는 동시에 사용 가능합니다.
